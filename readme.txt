LifterLMS is a Learning Management Plugin that helps you create and sell courses online without coding. With Rebrand LifterLMS, you can white label the plugin to change the name and description, while linking it to your own website instead of the developerâ€™s. It also allows you to choose your own primary color for buttons and links, and add your own logo.

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress(This will add new Sub Menu in admin panel's side menu name Rebrand` under LifterLMS menu).
3. Go to LifterLMS > Rebrand
